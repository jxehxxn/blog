## 캡스톤 옵션

### Option A: 사내 GoCD Platform
- HA 서버 + 외부 DB.
- 사내 secret plugin 자체 개발.
- yaml template 강제 + lint CI.
- Backstage 통합 (API).
- 100+ pipeline 운영.
- SOC 2 매핑 + audit dashboard.

### Option B: Source Contribution
- GoCD GitHub fork.
- good-first-issue 1개 해결.
- PR submit + 머지.

### Option C: Migration
- Jenkins 100+ job → GoCD 6개월 plan + 실행.
- 결과 retrospective.

## 평가
- Option A: 사내 사용자 만족도.
- Option B: PR 머지.
- Option C: migration 완료율 + 사고 0.

## 마무리

이 심화까지 마치면 빅테크 GoCD 운영팀의 senior engineer로 즉시 일할 수 있는 수준입니다.
- HA + 성능 + 운영 풀스택.
- Plugin Java 직접 개발.
- REST API + Backstage 통합.
- Governance + compliance.
- 마이그레이션 plan.

GoCD의 가장 큰 가치 = **CD 흐름 가시화 + audit + manual approval first-class**. 이게 필요한 환경 (금융/Healthcare/legacy enterprise)에서 senior로서의 가치 폭발.
